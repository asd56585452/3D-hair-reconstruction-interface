3D Strand-Based Hair Generation and Editing System

Motivation

With the advancement of AI technology, many virtual hairstyle try-on applications have emerged. However, these tools are limited to image-level editing and lack the capability for 3D reconstruction from different angles. To address these challenges, we have developed a system that efficiently generates interactive 3D hair models, simplifying the creation process and reducing production time.

Demo Video

Related Work

The open-source generative model HairStep, presented at CVPR 2023, uses U-Net to convert input images into a Depth Map and a Strand Map. A 2D-to-3D U-Net-like architecture is then employed to predict the hair’s Orientation field and Occupancy field.

System Function

Our system allows users to upload an image, edit the hair mask, and generate a 3D hair model within 10 minutes. The model can then be fine-tuned in Blender and exported as an Alembic file for use in Unity with real-time interaction.

System Overview

Conclusion

Our system provides a quick and efficient method to generate 3D hair models with minimal technical expertise. It is designed to be user-friendly for game developers looking to integrate realistic hair models into interactive environments.
